WHAT'S GOOD

JAVASCRIPT / CSS / HTML


What's Good is a content-based social media MERN stack application. Users rate and review music, movies, games, and books to add to their media collections, as well as like and comment on their friends' media. Also features a built-in message inbox for direct messaging. Developed in a team of four.

Responsible for: managing GitHub and Heroku deployment; games media's CRUD functions and Mongo schema; final design of React components; customn CSS; implementing dark mode; migrating mLab to MongoDB Atlas.
