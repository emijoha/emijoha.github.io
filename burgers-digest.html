BURGER'S DIGEST

TSQL / JAVASCRIPT / CSS / HTML


A 'Bob's Burgers' inspired web application that lets users input the names of burgers they'd like to eat, or click for a random burger from the show. Burger names are then displayed on a 'Burgers Served' or 'Burgers Digested' chalkboard-style menu depending on whether they've been 'eaten/digested' by the user. Burgers can also be deleted.

Solo project using Node, Express, Handlebars, two mySQL schemas, and a custom ORM.
